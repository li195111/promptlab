# PromptLab ⚡

> Free AI prompt management and testing tool. Runs entirely in your browser.

**[Try PromptLab →](https://li195111.github.io/promptlab/)**

## Features

- **Prompt Library** — Store, organize, and search prompts by category
- **Variable Detection** — Auto-detects `{variables}` and creates input fields
- **Multi-Model Testing** — Test prompts across OpenAI, Claude, and Gemini
- **BYOK** — Bring Your Own Key. Your API keys stay in your browser (localStorage)
- **Test History** — Track results, compare responses side-by-side
- **Import/Export** — Backup and share prompts as JSON
- **Zero Setup** — No signup, no server, no installation. Just open and use.

## Supported Models

| Provider | Models |
|----------|--------|
| OpenAI | GPT-4o, GPT-4o-mini, o3-mini |
| Anthropic | Claude Sonnet 4, Claude Haiku 4.5 |
| Google | Gemini 2.0 Flash, Gemini 2.5 Pro |

## How It Works

1. Open [PromptLab](https://li195111.github.io/promptlab/)
2. Create a prompt with `{variables}`
3. Enter your API key (stored locally, never sent to any server)
4. Click "Run Test" — see results instantly
5. Compare results across different models

## Privacy

- **No backend** — everything runs client-side in your browser
- **No tracking** — no analytics, no cookies, no data collection
- **API keys** are stored in localStorage only, never transmitted anywhere except directly to the AI provider
